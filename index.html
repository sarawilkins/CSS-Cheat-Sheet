Common CSS Properties

Property Modifies Additional Info
color foreground: 'color' or background-color named, hexadecimal, RGB, or HSL/HSLA
font-family typeface web safe, font stacks
font-size size of font takes values in px
font-weight bold, normal 0-1000; increments of 100
text-align align text to parent element left, center, right, justify
background-image background of an element as an image include relative or specific URL
border border of an element (width px, style i.e., solid, color)
padding space between content and border in px: 4, 3, or 2 values
margin space between border and outside edge of element same syntax as padding
overflow set on a parent; how to render child elements hidden, scroll, visible
position can have one of five values static(default), relative, absolute, fixed, sticky
z-index controls the depth of elements on the page uses integer values; does not work on static elements
